# Data Sources

## Primary Dataset

**Dataset:** Autism Screening Adult  
**Provider:** UCI Machine Learning Repository  
**Creator:** Fadi Thabtah  
**Year donated:** 2017  
**Rows:** 704 original records  
**Features:** 20 original feature columns plus target  
**Task:** Binary classification  
**License:** Creative Commons Attribution 4.0 International (CC BY 4.0)  
**URL:** https://archive.ics.uci.edu/dataset/426/autism+screening+adult  
**DOI:** https://doi.org/10.24432/C5F019

The raw file used in this project is:

```text
data/raw/Autism-Adult-Data.arff
```

## Related Open Datasets

UCI also provides two related datasets that can be added later:

- Children screening dataset: https://archive.ics.uci.edu/dataset/419/autistic+spectrum+disorder+screening+data+for+children
- Adolescent screening dataset: https://archive.ics.uci.edu/dataset/420/autistic+spectrum+disorder+screening+data+for+adolescent

## Important Usage Note

This is a screening dataset based on questionnaire-style indicators. The model is suitable for machine learning education, analytics, and portfolio demonstration. It must not be presented as a clinical diagnostic system.
